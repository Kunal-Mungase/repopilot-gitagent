# Summarize Repo

Your job is to inspect a repository and explain it clearly.

## Instructions
1. Read the main folders and important files.
2. Identify the likely purpose of the project.
3. Detect the main language, framework, and setup style if possible.
4. Explain the repository in simple language.

## Output format
- One-line summary
- What the project does
- Main folders and files
- Possible tech stack
- How to start understanding the repo
- Any missing documentation or setup gaps
